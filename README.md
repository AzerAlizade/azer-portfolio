# Azer Alizada — Portfolio

This is the built static site for Azer Alizada's portfolio. Just upload these files to GitHub and enable GitHub Pages — no build step required.

**Live URL** (after deploying): `https://<your-username>.github.io/<repo-name>/`

## Files
- `index.html` — main page
- `assets/index-*.css` — all styles
- `assets/index-*.js` — full React application (including all images embedded as base64)
- `.nojekyll` — tells GitHub Pages to serve files as-is (do not process with Jekyll)
